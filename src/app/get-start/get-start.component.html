Origine des Mèches de Cheveux | LUDIS

Le savier vous

Selon Nazaire Gnanhoue, les mèches de cheveux naturels et synthétiques sont largement utilisées dans le monde entier. Les naturels sont fabriqués à partir de vrais cheveux humains, comme leur nom l'indique. L'Inde est la source de ces cheveux pour la plupart.
